## 🚀 How to Use

### For Leadership Presentation

1. **Start Server** (1 min)
   ```bash
   cd web
   python start_server.py
   ```

2. **Open Browser** (auto or manual)
   - Double-click `index.html`
   - Or file:// URL

3. **Run Demo** (15 min)
   - Follow `DEMO-PRESENTATION-GUIDE.md`
   - Click demo buttons
   - Explain metrics
   - Show architecture

### For Development/Testing

1. **Test API**
   ```bash
   python test_api.py
   ```

2. **Start Server**
   ```bash
   python api_server.py
   ```

3. **Access Docs**
   - Swagger: http://localhost:8000/api/docs
   - ReDoc: http://localhost:8000/api/redoc
